Library is an open source multiplayer gaming platform built around classic card games. It combines real-time gameplay, accounts, leaderboards, chat, and constantly expanding games into one place.

see it running → view source on GitHub @TheOrangeCow

// about

Library started as a project to bring classic card games into a single self-hosted multiplayer platform. The goal was to create something that feels like a real online games platform while keeping the code open and customisable.

It is actively developed with new games, features and improvements being added regularly.
